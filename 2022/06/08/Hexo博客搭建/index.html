Hexo博客搭建

手把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo_哔哩哔哩_bilibili

1.下载安装node.js

Node.js 中文网 (nodejs.cn)

安装完成后，查看版本
npm -v
node -v

2.换阿里的源

npm install -g cnpm --registry=https://registry.npm.taobao.org

完成cnpm的安装，查看版本
cnpm -v

3.安装hexo博客框架

cnpm install -g hexo-cli

4.新建一个hexo博客

  • 新建文件夹Blog(任意一个名字)
  • 在该目录下打开终端输入hexo init完成博客初始化生成

5.在本地启动博客

hexo server or hexo s
