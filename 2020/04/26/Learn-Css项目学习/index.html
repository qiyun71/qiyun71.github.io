1.1 布局流程

为了提高网页制作的效率布局时通常有一下的流程

  • 必须确定页面的版心（可视区），测量可得
  • 分析页面中的行模块，以及每个行模块中的列模块，页面布局（一行行的罗列而成）
  • 制作HTML的结构，先结构，后样式
  • 运用盒子模型的原理，通过div+css布局来控制网页的各个模块
