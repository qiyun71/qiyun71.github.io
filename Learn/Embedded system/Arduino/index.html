1. 入门

  • 数字IO

输入设备 Input

  • 开关功能:某种状态下接通电路
    类型:数字输入(开或关)

  • 电位计功能:调节旋转程度
    类型:模拟输入(旋转不同值不同)

  • 蓝牙从功能:无线接收数据
    类型:串口输入(向Arduino发送信息)

输出设备 Output

  • LED功能:指示亮或灭
    类型:数字输出

  • 马达功能:转动
    类型:模拟输出(调整转速)

  • 蓝牙主功能:无线发送数据
    类型:串口输出

2.9 电位器模拟输入

电位器，旋钮改变R1，R2比值
R1 + R2 = 固定值

将5V分为1024份， 2.5V 对应数值 512
引脚输入范围可以使用analogReference()进行调整
